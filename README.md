# GREADER

Rewriting in Go of ncarlier nunux-reader

Features:

 * Dynamic GUI thanks to AngularJS
 * Responsive Web Design
 * OPML import/export
 * Manage subscriptions
 * Save articles
 * Keep article as not read
 * RESTFul JSON API
